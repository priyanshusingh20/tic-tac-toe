# 🎮 Tic Tac Toe AI (Web App)

A modern, responsive **Tic Tac Toe game with AI** built using HTML, CSS, and JavaScript.
Play against an intelligent AI with multiple difficulty levels and enjoy smooth animations and a clean UI.

## ✨ Features

* 🧠 **AI Opponent (Minimax Algorithm)**
* 🎯 **Difficulty Levels**

  * Easy (Random moves)
  * Medium (Balanced)
  * Hard (Unbeatable AI)
* 🎨 **Modern UI**

  * Animated gradient background
  * Glassmorphism design
  * Smooth hover & click effects
* 📱 **Responsive Design**

  * Works on mobile, tablet, and desktop
* 🏆 **Scoreboard**

  * Tracks wins, losses, and draws
  * Data stored using LocalStorage
* 🤖 **AI Thinking Animation**
* 🔄 Restart & Reset Score options

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3** (Animations, Flexbox, Grid)
* **JavaScript (Vanilla JS)**
* **LocalStorage API**

## 🧠 How AI Works

The AI uses the **Minimax Algorithm**:

* Evaluates all possible moves
* Chooses the optimal move
* Makes the AI unbeatable in Hard mode
